# Figma exports (referência)

Prints/exports do Figma pra eu montar o layout. **Não vão pro site** — só referência.

```
desktop/   → versão desktop de cada seção
tablet/    → versão tablet
mobile/    → versão mobile
```

## Como nomear
Prefixa com número da ordem + nome da seção. Mesmo nome nas 3 pastas = mesma seção.

```
desktop/01-hero.png
tablet/01-hero.png
mobile/01-hero.png
```

## Diferença pras imagens reais
- `assets/figma-exports/` → **referência** (screenshots do design)
- `public/images/` → **assets reais** usados no site (logos, fotos, ícones exportados)
